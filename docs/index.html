Quick Start

Up and Running in 3 Steps

01
Own Bloodshed on Steam

The mod requires the base game. Find it on Steam — search Bloodshed by com8com1.

02
Run the Installer

Download BloodshedModToolkitInstaller.exe from the latest GitHub release. It auto-detects your game path and installs BepInEx + the mod.

03
Launch & Press F5

Start Bloodshed. Press F5 in-game to open the mod menu. On first launch, BepInEx will generate interop assemblies — wait 30 s.
